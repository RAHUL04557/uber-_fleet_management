Add side-view vehicle images here with these exact names:

- uberx_side.png
- uberxl_side.png
- suv_side.png
- black_side.png
- moto_side.png

You can also use .jpg, .jpeg, or .webp instead of .png.

The Streamlit app will automatically use these files when present.
If a file is missing, the app falls back to the built-in side-view illustration for that vehicle.
